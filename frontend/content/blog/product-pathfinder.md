**한 줄 요약 —** 개발자만 가능했던 시스템 연동을, 누구나 가능한 실행 체계로 바꿉니다.

## 3단계로 끝내는 레거시 통합

1. **로그인** — 평소처럼 로그인하면 안전한 보안 세션이 준비됩니다
2. **API 스캔** — 화면을 쓰던 대로 움직이면 오가는 모든 API를 추적해 명세로 정리합니다
3. **도구 등록** — 버튼 하나로 Workflow 캔버스의 도구가 되어 누구나 사용합니다

## 무엇이 달라지나

API 문서가 없어도 화면 조작만으로 실제 호출 스펙을 캡처하고, 재사용 가능한 도구로 즉시 변환합니다. IT 의존 없이 현업이 직접 시스템을 연결합니다.
